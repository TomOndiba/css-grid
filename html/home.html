Facebook Login with Flexbox

A responsive login screen based on the work of Fred Nerby.

The goal was to convert one of his prototype screens into a working HTML/CSS template.

The template utilises SVG graphics for the logo, CSS Flexbox for layout, HTML5 form validation routines, a custom web font, CSS transitions and shadows for UI effects and media queries to handle changes in viewport size.
